NOTE FOR PITBOX DISTRIBUTION
-----------------------------
PitBox uses the `zeroc-ice` Python package at RUNTIME to communicate
with an already-running Mumble Server (murmur) over ICE on localhost.

CURRENT DEPLOYMENT (Fastest Lap internal only):
  PitBox does NOT bundle zeroc-ice into its installer.
  The zeroc-ice package is installed separately into the Python
  environment on the host machine (pip install zeroc-ice).
  This keeps PitBox's own distribution separate from the GPL-2.0
  Ice package.

FUTURE DISTRIBUTION NOTICE:
  If PitBox is ever distributed to external venues as a commercial
  product WITH zeroc-ice bundled, a commercial Ice license from
  ZeroC must be obtained, OR the zeroc-ice dependency must remain
  a separately installed external component that the end user installs.
  Do not bundle zeroc-ice inside a closed-source PitBox installer
  without legal review or a commercial ZeroC license.
